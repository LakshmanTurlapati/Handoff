<objective>
Replace manual daemon identity envs with a secure local bootstrap contract.

Purpose: After the package surface exists, the next blocker is auth and local state. The bridge currently has no durable hosted bootstrap identity and therefore can only start when an operator injects raw env values and a server signing secret.

Output: A durable bridge-installation record, a bridge bootstrap token returned from pairing confirmation, a hosted connect-ticket minting route, and secure local bootstrap state persisted by the CLI.
</objective>
