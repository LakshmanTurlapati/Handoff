Validate the `/handoff` path against approval, sandbox, and bridge-boundary constraints.

Output: Final fail-closed command copy and helper behavior, hosted ownership validation for handoff minting, and regression tests that prove `/handoff` remains a thread-bound remote-window feature rather than a broader shell surface.
</objective>
